Rules that make it a system

Chroma budget: ≤1 saturated element per region at rest; only meters + live mod-rings stay saturated. · Glow is a verb: record halo + mod activity only — nothing else glows. · Success = ACID (no second green). No cyan/magenta fills — muted cyan exists only as the color-category 3px tick. · Focus: 2px acid outline, offset 1px, everywhere. Handles visible at rest (≥6px painted). · Motion: 120/180/140ms, cubic-bezier(.2,0,0,1), transform/opacity only; one sanctioned RGB-split flicker (≤120ms) on render-complete / destructive-confirm. · Governance: components → semantic → primitives; hardcoded hex = CI ratchet failure. · Density: 24px rows, 28px panel headers, 18px param rows — every pixel is data. · Device voice: lowercase mono (pixel_sort); dialogs stay sentence-case Sans. · Boot line: one typed mono line on launch, reduced-motion safe.
