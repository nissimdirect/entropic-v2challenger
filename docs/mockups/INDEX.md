## Verification notes

All six new mocks were checked for:
- **Existence & size** — all present, 30.8 KB–54.8 KB (well over the 8 KB floor).
- **Zero external requests** — grepped for `http(s)://` in `src=`/`href=` attributes, `@import`, and `url(...)`, plus explicit checks for Google Fonts/CDN/jsdelivr/unpkg references. All clean; only `data:` URIs and in-page `#anchor` hrefs are used.
- **Numbered callout annotations** — each file uses a distinct callout convention (`.pin`, `.co`, `.callout`/`.num`, `.callout-row`) with numbered markers tying UI elements to PRD/plan/packet decisions.
- **Visually plausible structure** — each file has a real `<style>` block with `:root` custom-property tokens and dozens of `<div>` elements forming actual panel/card/grid layouts, not stub placeholders.
